# Iris Flower Classification

## Overview
This project is developed as part of the CodeAlpha Data Science Internship.

The machine learning model predicts the species of an Iris flower based on four flower measurements:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
- Random Forest Classifier

## Dataset
- Iris Dataset from Scikit-learn

## Accuracy
Approximately 100% (may vary slightly).

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

## Features

- Data Loading
- Data Analysis
- Data Visualization
- Model Training
- Accuracy Evaluation
- User Input Prediction
